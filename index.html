PERSONAL PROJECT

As a passionate game developer, I undertook a personal project to create a third-person combat system inspired by Dark Souls and Assassin's Creed, focusing on delivering responsive and engaging gameplay. Alongside this, I also worked on developing a 2D RPG and a tower defense game, exploring different genres and gameplay mechanics. These projects allowed me to deepen my skills in player input handling, state machines, combat systems, enemy behavior, and camera controls, while also strengthening my ability to design varied gameplay experiences across multiple styles of games.

I created a 3rd person combat system similar to Dark Souls or Assasin's creed. In this project I put the Player Input, State machines, Attack combo,dodging, target locking, cinemachine camera options.

Mix between a City-Builder and Tower-Defense game. The player has the ability to place buildings and those buildings gather resources. There's three starting resource types, Wood, Stone and Gold, although everything is set up in such a way that you can easily add more Resource Types. The Resource Generation is based on proximity so a Wood Harvester needs to be placed near Trees in order to gather Wood. The Buildings take some time to be Constructed and it shows that progress with a nice visual element and a really awesome Shader effect. Then we also have some Enemies, they are simple but still look great. Over time more and more enemies will be spawned until the player can't take it anymore.

I developed a versatile player controller using a finite state machine, built levels with parallax effects, set up camera controllers, designed enemies to fight against, and created an engaging combat system featuring mechanics like dashes, counterattacks, skills, and more. From there, I moved on to building more advanced systems to turn the project into a fully playable RPG. I added items, crafting mechanics, character stats such as critical hits, evasion, magic damage, elemental effects, and more. I also created UI elements to manage all these systems and developed a save system to track player progress, checkpoints, inventory, skills, experience, and currency.
